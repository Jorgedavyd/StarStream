## Example
```python
from starstream import ACE, DSCOVR
from starstream.utils import DataDownloading
from datetime import datetime

if __name__ == '__main__':
    DataDownloading(
        [
            ACE(),
            DSCOVR()
        ],
        (datetime(2014, 12, 12), datetime(2014, 12, 30))
    )

```
